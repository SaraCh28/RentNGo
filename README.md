RentNGo is a full-stack car rental web application that allows users to browse, book, and manage cars easily. It has a modern, responsive frontend built with Next.js, and a robust backend powered by NestJS.

Features

Browse available cars with details

Rent cars for specific dates

Admin can add, update, or delete car listings

Clean and modular code structure

REST API for frontend-backend communication

Tech Stack

Frontend: Next.js, React, JavaScript, CSS

Backend: NestJS, Node.js, TypeScript

Database: PostgreSQL

Installation & Running Locally

Backend
Open terminal and navigate to backend:
cd backend


Install dependencies:
npm install


Run backend server:
npm run start:dev


Frontend
Open terminal and navigate to frontend:
cd car_rental


Install dependencies:
npm install


Run frontend server:
npm run dev


Open your browser at http://localhost:3000
 to use the app.
